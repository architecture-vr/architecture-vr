# ArchitectureVR
ArchitectureVR is A Google Cardboard visualization platform. It's goal is to allow 3D models to be imported and experienced as in real life.

### Installation
Clone this repository
Open it up in Android Studio
Run Gradle to fetch dependencies
Enjoy! 

### Technology
ArchitectureVR is built upon the open-source Android 3D engine [Rajawali](https://github.com/Rajawali/Rajawali) and its sibling project [RajawaliVR](https://github.com/Rajawali/RajawaliVr)

### Caveats 
* If models are too big or memory intensive, the program will crash
* Textures are not yet loading properly
* Etc.

Please feel free to contribute!
